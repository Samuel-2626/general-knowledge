# Introduction to Software Startups

## What is a software product?

- Code that solves a problem
- Input and outputs
- User packaging
- Can generate cash

## A company's priorities

- Increase profit
- More customers
- More $ per customer
- Improve product
- New products
- New business area
- Acquire others

## A startup's priorities

- Don't die
- Find some users
- Get $ from users
- Create a product
- Find a business area
- Get acquired

## The first version

- Minimum viable product
- Identify early adopters
- Build quickly
- Design for Learning
- No barriers to use
- Aim to fail fast

For people, UI = product
